## FILES STUFF

Versión 0.0001

Pequeña aplicación para realizar tareas sobre archivos.
Creado con Python y Flet.

## Características

- Buscar archivos duplicados
- Oraganizar archivos en carpetas (Imágenes, Documentos, etc ...)
- Renombrar archivos de forma masiva (renombra TODOS ... y SOLO ... los archivos deuna carpeta)
- Redimensión de imágenes.

### Actualizaciones

- Se ha modularizado el programa
- Se ha añadido la función de renombrado masivo de los archivos incluidos en una carpeta. No actúa sobre carpetas.
- Añadida la capacidad de redimensionar imágenes de una carpeta. Los archivos soportados son .jpg, .jpeg, .png, .bmp, .gif y .tiff.

## Ejecutar la aplicación

```
python3 run_app.py
```
